# YoClaw

YoClaw 是一个智能对话机器人框架，支持多种通信渠道接入。

## 快速开始

### 安装

从 [Releases](https://github.com/yockii/YoClaw/releases) 页面下载对应平台的可执行文件。

### 使用方法

详细的使用指南请参考 [Wiki - 使用指南](https://github.com/yockii/YoClaw/wiki/%E4%BD%BF%E7%94%A8%E6%8C%87%E5%8D%97)。

## 功能特性

- 支持多平台接入（飞书等）
- 可扩展的 Skill 系统
- 内置多种工具支持
